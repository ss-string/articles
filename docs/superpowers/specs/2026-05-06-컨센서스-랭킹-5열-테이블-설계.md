# 컨센서스 랭킹 5열 테이블 설계

## 목표

컨센서스 괴리율 랭킹 테이블을 다음 5열 구조로 고정한다.

```text
종목 | 현재가 | 적정가 | 갭 | 적정가 대비 현재가
```

기존 마지막 컬럼에 함께 표시되던 `지난 1개월 대비 컨센서스 증가` 배지는 제거한다. 마지막 컬럼은 현재가가 적정가 대비 어느 수준인지 보여주는 막대와 비율 표시에만 집중한다.

## 컬럼별 표시

### 종목

종목명과 종목코드를 기존처럼 표시한다. 종목코드가 없으면 종목명만 표시한다.

### 현재가

현재 가격을 원화 형식으로 표시한다.

### 적정가

기존 `적정주가` 컬럼명을 `적정가`로 줄이고, 적정가를 원화 형식으로 표시한다. 갭 금액은 이 컬럼에 보조 텍스트로 표시하지 않는다.

### 갭

갭 컬럼은 라벨 없이 두 줄로 표시한다.

```text
+45,000원
+81.8%
```

첫 줄은 `적정가 - 현재가` 금액이다. 둘째 줄은 기존 괴리율이다. 양수는 기존 녹색 계열, 음수는 기존 붉은 계열을 사용한다.

### 적정가 대비 현재가

현재가 비율 막대를 표시한다.

```text
현재가 비율 = 현재가 / 적정가 * 100
```

- 100% 이하: 막대 너비를 실제 비율만큼 채우고, 오른쪽에 `55.0%`처럼 표시한다.
- 100% 초과: 막대는 100%까지만 채우고, 오른쪽에 `초과 130.0%` 배지를 표시한다.

## 제거 항목

- 마지막 컬럼 아래의 `지난 1개월 대비 컨센서스 증가` 배지
- 마지막 컬럼에서 컨센서스 증가율을 나타내던 텍스트

## 반응형 동작

모바일 카드형 row에서도 같은 컬럼 의미를 유지한다. 모바일 라벨은 다음처럼 짧게 표시한다.

- 현재가
- 적정가
- 갭
- 적정가 대비 현재가

텍스트와 배지는 같은 셀 안에서 겹치지 않아야 한다.
