# 매크로 레짐 카드뷰와 상세 팝업 설계

## 목적

Supabase의 `macro_regime_decisions` 테이블을 기반으로 한국과 미국의 최신 매크로 레짐 판단을 카드로 보여준다. 사용자는 카드에서 날짜, 국가, 레짐 판단, 4개 축 판단을 빠르게 확인하고, 카드를 선택하면 화면 전환 없이 팝업에서 상세 설명 전체를 읽는다.

## 데이터 조회 기준

지면은 `macro_regime_decisions` 테이블을 조회한다. 최신 판단은 `updated_at` 내림차순 기준으로 결정한다.

국가별로 다음 쿼리 의미를 사용한다.

- `market = 'KR'` 조건에서 `updated_at`이 가장 최신인 행 1건
- `market = 'US'` 조건에서 `updated_at`이 가장 최신인 행 1건

카드와 팝업에서 사용하는 주요 필드는 다음과 같다.

- `run_date`: 카드와 팝업에 표시하는 날짜
- `market`: `KR`, `US` 국가 라벨
- `regime`: 최종 레짐 판단
- `summary`: 팝업 요약
- `axis_assessments`: 성장, 물가, 통화, 유동성 축별 판단과 설명
- `key_indicators`: 핵심 지표 목록
- `asset_implications`: 자산 영향
- `risk_factors`: 리스크
- `content_md`: 전문

팝업 상단에는 `as_of_date`, `updated_at 최신` 같은 보조 문구를 노출하지 않는다. 날짜는 `run_date`를 `YYYY.MM.DD` 형식으로 변환해 표시한다.

조회 결과가 없는 국가는 카드 위치를 유지하고 빈 상태 문구를 표시한다. 두 국가 모두 조회에 실패하면 섹션 전체에 오류 상태를 표시한다.

## 화면 구조

기존 포트폴리오 대시보드 스타일을 유지한다.

- 좌측 사이드바와 모바일 상단 내비게이션은 기존 앱 셸을 사용한다.
- 내비게이션에는 `매크로 레짐` 항목을 추가하고 `#macro-regime`으로 이동한다.
- 매크로 레짐 지면은 기존 `dashboard-section` 계열 어두운 패널 안에 배치한다.
- 제목은 `최신 매크로 레짐 판단`으로 둔다.
- 설명 문구는 국가별 최신 판단을 먼저 확인하고, 카드 선택 시 상세를 팝업으로 읽는다는 정도로 짧게 유지한다.

카드는 하나의 행에 두 개를 배치한다.

- 왼쪽 카드: `KR`
- 오른쪽 카드: `US`
- 각 카드는 데스크톱에서 절반 폭을 차지한다.
- 모바일에서는 카드가 세로로 쌓인다.

카드에는 다음 정보를 표시한다.

- 국가 라벨
- 날짜
- 레짐 판단
- 짧은 축 요약 문장
- 성장, 물가, 통화, 유동성 4개 축의 판단값

## 상세 팝업

카드를 선택하면 화면 전환 없이 모달 팝업을 연다. 팝업은 기존 어두운 대시보드 톤을 유지한다.

팝업 상단에는 다음만 표시한다.

- 국가 라벨
- 날짜
- 레짐 판단
- 닫기 버튼

팝업 본문은 좌측 목차와 우측 스크롤 본문으로 구성한다.

- 요약
- 4개 축 판단
- 핵심 지표
- 자산 영향
- 리스크
- 전문

본문만 스크롤되며, 팝업 헤더와 닫기 버튼은 고정한다. 모바일에서는 좌측 목차를 상단 가로 칩 형태로 바꾸고 팝업은 거의 전체 화면으로 표시한다.

## 4개 축 판단 표시

팝업의 4개 축 판단은 스퀘어 그리드로 묶지 않는다. 각 축을 하나씩 세로로 나열한다.

각 축은 다음 순서를 맞춘다.

```text
축명 - 판단 - 컨피던스
설명
```

예시는 다음과 같다.

```text
성장 - 판단 neutral - 컨피던스 0.69
GDP YoY, 비농업고용 증감, 산업생산 YoY의 signal_score는...
```

축 순서는 성장, 물가, 통화, 유동성으로 고정한다.

## 핵심 지표 표시

핵심 지표는 표 형태로 표시한다.

표에는 다음 값을 표시한다.

- 지표명
- 값과 단위
- 추세
- 출처
- 해석

`trend_3m` 값은 그대로 `up`, `down`, `flat`으로 노출하지 않는다. 다음 텍스트와 기호로 변환한다.

- `up`: `↑ 상승`
- `down`: `↓ 하락`
- `flat`: `→ 보합`

추세 셀에는 배경색을 넣지 않고 텍스트 색만 적용한다.

- 상승: 청록 계열 텍스트
- 하락: 빨강 계열 텍스트
- 보합: 노랑 계열 텍스트

## 전문 표시

`content_md`는 팝업의 `전문` 섹션에 표시한다. 마크다운 전체 기능을 구현하지 않고, 현재 데이터에서 필요한 제목, 문단, 목록, 표 수준만 읽기 좋게 변환한다.

마크다운 표 변환이 범위를 과하게 키우면 초기 구현에서는 `content_md`를 문단 중심으로 표시하고, 구조화 데이터 섹션이 이미 같은 정보를 제공하도록 한다.

## 로딩, 빈 상태, 오류 상태

다음 상태를 표시한다.

- Supabase 조회 중 로딩 상태
- 특정 국가의 최신 행이 없을 때 해당 카드의 빈 상태
- 두 국가 모두 표시할 수 없을 때 섹션 빈 상태
- 환경변수 누락 또는 Supabase query 실패 시 오류 상태

오류 상태는 비밀값을 노출하지 않는다.

## 접근성

- 카드는 버튼으로 구현하고 키보드 선택을 지원한다.
- 모달은 `role="dialog"`와 `aria-modal="true"`를 사용한다.
- 모달 제목은 `aria-labelledby`로 연결한다.
- 닫기 버튼, 배경 클릭, `Escape` 키로 닫을 수 있게 한다.
- 모달이 열려 있는 동안 포커스가 모달 밖으로 빠져나가지 않게 한다.

## 테스트

다음 항목을 테스트한다.

- `macro_regime_decisions` 행을 국가별 최신 판단 모델로 정규화한다.
- `updated_at` 기준 최신 KR, US row를 조회한다.
- 카드에 날짜, 국가, 레짐 판단, 4개 축 판단이 표시된다.
- 카드 선택 시 화면 전환 없이 팝업이 열린다.
- 팝업 상단에는 날짜만 표시하고 `as_of_date`, `updated_at 최신` 문구는 표시하지 않는다.
- 4개 축 판단이 축명, 판단, 컨피던스, 설명 순서로 표시된다.
- 핵심 지표의 `up`, `down`, `flat`이 `↑ 상승`, `↓ 하락`, `→ 보합`으로 표시된다.
- 자산 영향, 리스크, 전문 영역이 표시된다.
- 로딩, 빈 상태, 오류 상태가 표시된다.
- 전체 테스트와 프로덕션 빌드가 통과한다.

## 수용 기준

- `macro_regime_decisions`에서 KR과 US의 최신 판단을 각각 `updated_at` 기준으로 가져온다.
- 최신 판단 카드는 하나의 행에서 KR, US를 절반 폭으로 나란히 보여준다.
- 각 카드는 날짜, 국가, 레짐 판단, 성장, 물가, 통화, 유동성 판단을 표시한다.
- 카드 선택 시 같은 화면 위에 상세 팝업이 열린다.
- 팝업은 실제 행 데이터를 기반으로 요약, 4개 축 판단, 핵심 지표, 자산 영향, 리스크, 전문을 표시한다.
- 팝업 상단 날짜는 `run_date`만 표시한다.
- 4개 축 판단은 스퀘어 그리드가 아니라 축별 세로 목록으로 표시한다.
- 추세는 텍스트 색만 적용한 `↑ 상승`, `↓ 하락`, `→ 보합`으로 표시한다.
- 기존 포트폴리오 대시보드의 어두운 스타일과 시각 규칙을 유지한다.
