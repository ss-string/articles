# 컨센서스 현재가 비율 막대 설계

## 목표

컨센서스 괴리율 랭킹 테이블의 `지난 1개월 대비 컨센서스 증가` 컬럼에 있는 막대를 현재가와 적정주가의 상대 위치를 보여주는 그래프로 바꾼다. 사용자는 컨센서스 증가율 배지를 보면서도, 현재 주가가 적정주가 대비 어느 정도까지 도달했는지 한눈에 확인할 수 있어야 한다.

## 계산 기준

막대의 기준값은 다음 비율이다.

```text
현재가 비율 = 현재가 / 적정주가 * 100
```

현재가가 적정주가의 55%이면 막대는 55%만 채운다. 현재가가 적정주가 이상이면 막대는 100%까지만 채우고, 실제 비율이 100%를 넘었다는 표시를 별도로 둔다.

## 화면 동작

확정안은 `막대 + 초과 배지` 방식이다.

- 100% 이하: 녹색 계열 막대와 비율 텍스트를 표시한다.
- 100% 초과: 막대를 100%까지 붉은 계열로 채우고, 오른쪽에 `초과 N%` 배지를 표시한다.
- 기존 `지난 1개월 대비 컨센서스 증가` 배지는 유지한다.
- 막대와 배지는 마지막 컬럼 안에서 줄바꿈되어도 겹치지 않아야 한다.

- `src/components/ConsensusTable.test.tsx`: 55%와 130% 사례를 검증한다.
