# 상세 progress 그래프 설계

## 목표

컨센서스 랭킹에서 종목을 선택했을 때 열리는 상세 팝업의 `가격 비교` 카드 progress 그래프를 테이블의 `적정가 대비 현재가` 그래프와 같은 기준으로 맞춘다.

현재 상세 팝업은 progress 막대 너비에 괴리율을 사용한다. 앞으로는 다음 비율을 사용한다.

```text
현재가 비율 = 현재가 / 적정가 * 100
```

## 확정 시안

확정안은 `B. 2지표 + progress 설명`이다.

가격 비교 카드 상단에는 2개 지표를 같은 높이의 타일로 표시한다.

- 현재가
- 적정가

좁은 화면에서 금액 줄바꿈이 커지는 문제를 줄이기 위해 `갭` 타일은 표시하지 않는다. 갭 금액은 progress 설명 문구에서만 표시한다.

2개 지표 아래에는 `적정가 대비 현재가` progress 영역을 표시한다.

- progress 제목: `적정가 대비 현재가`
- 오른쪽 값: `67.6%` 또는 `초과 130.0%`
- 막대 너비: 현재가 비율, 최대 100%
- 설명 문구: 적정가보다 낮으면 `적정가까지 N원 남았습니다.`, 높으면 `적정가를 N원 초과했습니다.`, 같으면 `현재가와 적정가가 같습니다.`

## 초과 상태

현재가 비율이 100%를 넘으면 막대는 100%까지만 채운다. 오른쪽 값은 테이블과 같은 `초과 N%` 배지로 표시한다.

## 용어 정리

상세 팝업의 가격 비교 영역에서는 `적정주가` 대신 `적정가`를 사용한다. 기존 설명 문구도 `적정가`로 맞춘다.
